# Cassette Sprite CSS Demo

웹 기준으로 스프라이트를 검증할 수 있게 mockup-sandbox 컴포넌트를 추가했다.

프리뷰 경로:

- `/preview/CassetteSpritePlayer`

파일:

- `artifacts/mockup-sandbox/src/components/mockups/CassetteSpritePlayer.tsx`
- `artifacts/mockup-sandbox/src/components/mockups/CassetteSpritePlayer.css`

필요 자산:

- `artifacts/mockup-sandbox/public/sprites/cassette-sprite.png`

현재 CSS는 사용자가 제안한 스프라이트 배치 방식을 기준으로 정리했다.
실제 스프라이트 PNG를 넣은 뒤 `background-position`과 `aspect-ratio`만 미세조정하면 된다.
