# Cassette Tape Music Player App

## 프로젝트 개요
React Native + Expo + TypeScript로 만드는 안드로이드 로컬 뮤직 플레이어.
빈티지 카세트 테이프 UI/UX가 핵심.

## 기술 스택
- React Native + Expo (TypeScript)
- expo-av (오디오 재생)
- expo-media-library (로컬 음악 파일)
- react-native-reanimated (애니메이션)
- react-navigation (화면 전환)

## 폴더 구조
src/
├── screens/
│   ├── PlayerScreen.tsx      # 메인 카세트 플레이어
│   └── LibraryScreen.tsx     # 음악 목록
├── components/
│   ├── CassetteTape.tsx      # 카세트 SVG 컴포넌트
│   ├── Spool.tsx             # 릴 애니메이션 컴포넌트
│   ├── ControlButtons.tsx    # 재생/정지/이전/다음 버튼
│   └── TrackLabel.tsx        # 카세트 라벨 (곡 제목 표시)
└── hooks/
└── useAudioPlayer.ts     # 오디오 재생 로직 훅

## 디자인 컨셉

## 구현 순서 (이 순서대로 진행해줘)

### Step 1 - 프로젝트 초기 세팅
- 필요한 패키지 설치 안내 (expo install 명령어)
- App.tsx에 react-navigation 기본 세팅
- 색상/폰트 상수 파일 생성 (src/constants/theme.ts)
- 완료되면 알려줘

### Step 2 - 오디오 훅 구현
useAudioPlayer.ts 훅 만들기:
- expo-media-library로 안드로이드 음악 파일 불러오기
- 권한 요청 처리
- expo-av로 재생/정지/이전곡/다음곡
- 현재 재생 위치 & 전체 길이 반환
- 재생 진행도 0~1 값 반환 (릴 애니메이션 연결용)
- 완료되면 알려줘

### Step 3 - 카세트 UI 컴포넌트
CassetteTape.tsx + Spool.tsx 만들기:
- 카세트 테이프 모양 SVG (react-native-svg 사용)
- 카세트 안에 왼쪽/오른쪽 릴 두 개
- 재생 중일 때 릴 회전 애니메이션 (react-native-reanimated)
- progress(0~1) props를 받아서:
  - 왼쪽 릴: progress에 따라 점점 작아짐
  - 오른쪽 릴: progress에 따라 점점 커짐
  - 실제 테이프가 감기는 느낌
- isPlaying props: true일 때만 회전
- 완료되면 알려줘

### Step 4 - 플레이어 화면
PlayerScreen.tsx 만들기:
- 중앙에 CassetteTape 컴포넌트
- 카세트 라벨에 현재 곡 제목 + 아티스트 표시
- 하단에 ControlButtons (이전/재생-정지/다음)
- 빨리감기/되감기 버튼 (길게 누르면 탐색)
- 재생 진행도 텍스트 표시 (현재시간 / 전체시간)
- 전체 배경 빈티지 톤
- 완료되면 알려줘

### Step 5 - 음악 목록 화면
LibraryScreen.tsx 만들기:
- 기기 내 음악 파일 목록 표시
- 곡 제목 / 아티스트 / 앨범 표시
- 곡 누르면 PlayerScreen으로 이동 + 재생 시작
- 빈티지 카세트 느낌 리스트 UI
- 완료되면 알려줘

### Step 6 - 마무리 디테일
- 재생/정지 시 햅틱 피드백 (expo-haptics)
- 카세트 케이스 디테일 (나사, 투명창 등) SVG 개선
- 앱 전체 색상 테마 통일 확인
- 안드로이드 권한 설정 확인 (app.json)
- 완료되면 알려줘

## 주의사항
- 모든 컴포넌트는 TypeScript 타입 명확히 지정
- 애니메이션은 반드시 react-native-reanimated 사용 (JS 스레드 애니메이션 금지)
- 각 Step 완료 후 반드시 알려주고 다음 Step 진행
